# Travel Trucks

## Опис проекту

**Travel Trucks** — це веб-додаток для пошуку, перегляду та бронювання кемперів. Він створений з використанням сучасних інструментів фронтенд-розробки, таких як React, Redux, TypeScript і Vite. Проект забезпечує швидку та зручну роботу завдяки легкій інтеграції з серверними API та оптимізованому рендерингу компонентів.

## Основні функції

- Каталог кемперів із фільтрацією за типом, обладнанням і локацією.
- Перегляд деталей кемпера.
- Можливість додавання кемперів до обраного.
- Бронювання кемперів через інтерактивну форму.
- Підтримка відгуків.

## Технології

Проект створено за допомогою таких технологій:

- **React** — для створення користувацького інтерфейсу.
- **Redux Toolkit** — для управління станом.
- **React Router** — для маршрутизації.
- **Vite** — для швидкої збірки та розробки.
- **TypeScript** — для статичної типізації.
- **Axios** — для запитів до API.
- **Styled Components** — для стилізації компонентів.

## Інструкції з установки та запуску

### 1. Клонуйте репозиторій


  - Демонстрація привітального тексту із заголовком "Campers of your dreams".

### **Фільтри**

- **Компонент:** `Filters.tsx`
- **Функціонал:**
  - Надання користувачеві можливості фільтрувати кемпери за типом, обладнанням та локацією.
  - Кнопка "Search" оновлює список кемперів на основі вибраних фільтрів.
- **Особливості:** Використовує Redux для управління станом фільтрів.

### **Картка кемпера**

- **Компонент:** `CamperCard.tsx`
- **Функціонал:**
  - Відображає інформацію про окремий кемпер (назва, ціна, рейтинг, локація).
  - Динамічне додавання/видалення з обраних.
  - Кнопка "Show More" перенаправляє на детальну сторінку кемпера.
- **Особливості:** Підтримує інтерактивність через Redux та адаптований стиль для коротких і довгих назв.

### **Сторінка деталей кемпера**

- **Компонент:** `CamperDetailsPage.tsx`

## **Управління станом**

Проект використовує **Redux Toolkit** для управління глобальним станом додатку. Це включає фільтри, список кемперів, обрані елементи, деталі кемпера та асинхронні операції для отримання даних.

### **Основні слайси:**

#### **1. Слайс `filtersSlice`**

- **Мета:** Управління станом фільтрів каталогу.
- **Стан (`state`):**
  - `filters`: Об'єкт із активними фільтрами.
  - `page`: Поточна сторінка каталогу.
  - `limit`: Кількість елементів на сторінку.
  - `totalPages`: Загальна кількість сторінок.
  - `loading`: Стан завантаження (true/false).
  - `error`: Інформація про помилку (якщо є).
- **Дії:**
  - `setFilters(filters)`: Встановлення нових фільтрів.
  - `setPage(page)`: Встановлення поточної сторінки.
  - `resetFilters()`: Скидання всіх фільтрів.
  - `setLocation(location)`: Встановлення локації для фільтрації.

#### **2. Слайс `campersSlice`**

- **Мета:** Управління списком кемперів, деталями та обраними.
- **Стан (`state`):**
  - `items`: Масив кемперів.
  - `loading`: Стан завантаження.
  - `error`: Інформація про помилку (якщо є).
  - `camperDetails`: Детальна інформація про вибраний кемпер.
  - `favorites`: Масив обраних кемперів.
  - `currentPage`: Поточна сторінка.
  - `totalPages`: Загальна кількість сторінок.
- **Дії:**
  - `toggleFavorite(camperId)`: Додавання або видалення кемпера з обраних.

## **Асинхронні операції**

Проект використовує `createAsyncThunk` з Redux Toolkit для виконання API-запитів.

### **Операції:**

#### 1. `fetchFilteredCampers`

- **Мета:** Отримання списку кемперів на основі фільтрів та пагінації.
- **Параметри:**
  - `filters`: Активні фільтри.
  - `page`: Номер сторінки.
  - `limit`: Ліміт елементів на сторінку.
- **Використання:** Запит надсилається на сервер із параметрами, і отримані дані оновлюють список кемперів у стані.

#### 2. `fetchCamperDetails`

- **Мета:** Отримання деталей окремого кемпера.
- **Параметри:** ID кемпера.
- **Використання:** Викликається при переході на сторінку деталей кемпера.

## **Селектори**

Для зручного доступу до даних у стані проект використовує селектори:

### **Селектори кемперів:**

- `selectCampers`: Повертає масив кемперів.
- `selectLoading`: Стан завантаження.
- `selectError`: Інформація про помилки.
- `selectCamperDetails`: Деталі обраного кемпера.
- `selectFavorites`: Масив обраних кемперів.

### **Селектори фільтрів:**

- `selectFilters`: Об'єкт активних фільтрів.
- `selectPage`: Поточна сторінка.
- `selectTotalPages`: Загальна кількість сторінок.
- `selectFilterLoading`: Стан завантаження фільтрів.
- `selectLocation`: Вибрана локація.
- `selectLimit`: Ліміт елементів на сторінку.
